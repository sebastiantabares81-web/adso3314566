08 - Relative Units

Specify a length relative to another length property.

  • Pixel (px)
  • M (em)
  • X (ex)
  • Root M (rem)
  • viewport units horizontal (vh)
  • viewport units with (vw)
  • large viewport (lvh|lvw)
  • small viewport (svh|svw)
  • dinamic viewport (dvh|dvw)

  • px: 15px
  • em: 1.5em
  • rem: 2rem
  • ex: 2.4ex
